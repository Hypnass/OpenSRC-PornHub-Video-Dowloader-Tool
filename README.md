Hello everyone,

I wrote a proper program in my free time using Opus that downloads videos from PH, so I thought I’d share it. It has both a graphical interface and can also be used from the command line. It downloads the highest available quality, including 4K.

Briefly, what it does:

* Automatically selects the best available quality, or you can choose manually, MP4/MKV
* Download queue, history, pause/resume are all included
* Downloads in the background, so the interface does not freeze
* Includes settings such as proxy, speed limit, and folder selection

Installation (EXE — easiest method)

1. Download the ZIP below
2. Extract it to a folder
3. Run PHDownloader.exe, or build it with build.exe and then run it

On first launch, it downloads FFmpeg automatically. You just need to wait for that to finish. Until then, videos will be downloaded as .ts files, which already play fine in VLC.
